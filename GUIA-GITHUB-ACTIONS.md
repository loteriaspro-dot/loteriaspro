# Como ativar a atualização automática do histórico

1. Crie um repositório no GitHub.
2. Envie todos os arquivos deste ZIP para o repositório.
3. Entre no repositório no GitHub.
4. Vá em **Actions**.
5. Clique em **I understand my workflows, go ahead and enable them**, se aparecer.
6. Abra o workflow **Atualizar histórico das loterias**.
7. Clique em **Run workflow** para rodar manualmente a primeira vez.
8. Aguarde finalizar.
9. Depois disso, a pasta `historico/` será preenchida com os arquivos JSON.

Depois da primeira execução, o GitHub Actions roda automaticamente todos os dias.
